科系履歷

資工系實習履歷範例：專題、GitHub、課程經歷怎麼寫

資工系大學生實習履歷寫法，包含專題、GitHub、課程、Side Project 與技能欄位範例。

資工履歷要先讓作品可被檢查

如果有 GitHub、作品集或 Demo，放在履歷上方。專題描述要包含技術、功能、你的貢獻和結果，不要只列框架名稱。

技能欄位要分層

可以分成 Programming、Frontend、Backend、Database、Tools。熟悉程度不足的工具不要硬塞，面試時被追問會更吃虧。

專題條列範例

Developed a course planning web app with login, filtering, and saved schedule features; implemented frontend components and connected API responses to user-facing views.

延伸閱讀

  • 教授推薦信邀請信範例：寄出前先整理好這些資料
  • 交換學生讀書計畫怎麼寫：動機、選校理由與返國規劃

常見問題

這份申請文件可以直接送出嗎？

不建議直接送出。本站提供的是架構與初稿，送出前應依照你的真實經歷、申請單位要求與截止日期再次修改。

沒有特殊經歷也可以使用這些模板嗎？

可以。大學生常見的課程、專題、社團、打工、志工與競賽都能轉換成申請文件素材，重點是寫出具體任務與成果。
