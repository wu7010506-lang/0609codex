# 大學生申請工具箱上線計畫

## 零成本上線

建議用 GitHub Pages 或 Cloudflare Pages。這個網站是純靜態 HTML/CSS/JS，不需要後端、不需要資料庫、不需要 API key。

## 上線前要改

1. 把 `sitemap.xml` 和 `robots.txt` 裡的 `https://example.com` 換成正式網域。
2. 網站名稱可以先維持「大學生申請工具箱」。
3. 若沒有自訂網域，先用免費子網域上線也可以。
